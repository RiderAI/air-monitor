# Air Monitor

Интерактивный веб-дашборд для CSV-логов мониторинга атмосферы с автономного Arduino-устройства.

Сервис показывает GPS-треки, оценочные поля загрязнений, мультисенсорные графики, MQ-калибровку по двум точкам и экспорт отфильтрованных данных. CSV-файлы наблюдений и JSON-пресеты калибровки обрабатываются локально в браузере; серверное хранение данных не используется.

Встроенный `MONITOR.CSV` является синтетическим демонстрационным набором данных для показа возможностей интерфейса.

## English

Air Monitor is a static browser dashboard for Arduino air-monitoring CSV logs. It visualizes GPS tracks, estimated pollution fields, sensor timelines, MQ calibration presets and filtered CSV exports.

All user data processing happens locally in the browser. No backend storage is used.

The bundled `MONITOR.CSV` is synthetic demo data.
