# Air Monitor

**Air Monitor** — интерактивная веб-платформа для анализа полевых данных атмосферного мониторинга, собранных автономными измерительными модулями.

Сервис предназначен для визуализации данных мониторинга с датчиков качества воздуха, GPS, ветра, температуры и влажности. Он объединяет геопространственный анализ, временные графики, калибровку газовых сенсоров и экспорт исследовательских срезов в едином интерфейсе.

## Возможности

- Интерактивная карта измерений с GPS-треком, цветовой индикацией выбранной метрики и оценочным полем загрязнений.
- Аналитика временных рядов для MQ-сенсоров, погодных параметров и дополнительных числовых каналов.
- Калибровка MQ-сенсоров по двум точкам с сохранением и загрузкой калибровочных профилей.
- Фильтрация по времени, сессиям измерений, качеству GPS, погодным данным и диапазонам значений.
- Экспорт отфильтрованных данных в CSV для последующей обработки и отчётности.
- Встроенный демонстрационный набор данных для быстрой оценки интерфейса и аналитического сценария.

## Сценарии применения

Air Monitor подходит для анализа данных мобильных и автономных систем экологического мониторинга: устройств на крыше автомобиля, дроне, стационарной точке наблюдения или экспериментальной измерительной платформе.

## English

**Air Monitor** is an interactive web platform for analyzing field atmospheric monitoring data collected by autonomous sensing modules.

It visualizes monitoring data from air-quality sensors, GPS, wind, temperature and humidity channels. The dashboard combines geospatial analysis, time-series visualization, MQ sensor calibration and filtered data export in a single interface.

## Key Capabilities

- Interactive measurement map with GPS tracks, metric-based coloring and estimated pollution fields.
- Time-series analytics for MQ sensors, weather channels and additional numeric data.
- Two-point MQ sensor calibration with reusable calibration profiles.
- Filtering by time range, measurement session, GPS quality, weather availability and metric values.
- Filtered CSV export for reporting and further analysis.
- Included demonstration dataset for evaluating the full analysis workflow.
